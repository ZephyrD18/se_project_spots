# Spots

An interactive front-end photo sharing interface built by **Issam Driwech**.

---

## Overview

Spots is a dynamic photo-sharing web application focused on clean UI, structured JavaScript, and strong user experience practices.

Users can:

- Edit their profile information
- Add new photo posts
- Like and delete posts
- Preview images in a modal view

This project emphasizes modular JavaScript, reusable form validation, and production-aware event handling.

---

## Live Features

- Edit profile modal with validation
- Add new post modal with real-time validation
- Like/unlike photo cards
- Delete photo cards
- Image preview modal
- Close modals with Escape key
- Close modals by clicking outside (overlay)
- Disabled submit button when inputs are invalid
- Fully responsive layout

---

## Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Responsive layout & styling
- **Vanilla JavaScript (ES6)** – DOM manipulation & modular logic

---

## Key Concepts Demonstrated

- Template cloning for dynamic rendering
- Event listener management
- Modal state control
- Form validation using a configuration object
- Separation of concerns
- Preventing memory leaks (cleaning up event listeners properly)

---

## Future Improvements

- Connect to a backend API for persistent storage
- Add animations to modal transitions
- Refactor into ES modules
- Add unit tests for validation logic
